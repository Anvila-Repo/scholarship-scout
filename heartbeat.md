# Heartbeat

## Daily Cadence
- **Phase 1: Scout (Morning)**: Query target databases, keywords, and sources for newly published scholarships.
- **Phase 2: Filter & Vet (Midday)**: Pass discovered links through security and viability filters. Extract meta-data (Amount, Eligibility, Deadline).
- **Phase 3: Compose (Afternoon)**: Generate draft social posts for validated opportunities.
- **Phase 4: Queue (Evening)**: Present ready-to-publish drafts to the user for final approval or push to the social media scheduler queue.
